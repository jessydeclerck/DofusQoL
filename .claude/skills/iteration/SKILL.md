Implémente l'itération $ARGUMENTS du projet DofusManager.

Workflow obligatoire :
1. Lis docs/specs.md pour comprendre les features de cette itération
2. Lis CLAUDE.md pour le contexte et les conventions
3. Propose un plan d'implémentation AVANT de coder (attends ma validation)
4. Après validation, implémente fichier par fichier
5. Écris les tests pour chaque composant
6. Vérifie que dotnet build et dotnet test passent
7. Fais un commit atomique par composant
8. Mets à jour CLAUDE.md (état d'avancement de l'itération)
